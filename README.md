BBattleASD is a large-scale arena infrastructure plugin focused on creating cinematic, progression-based and highly configurable boss battle experiences.

The plugin was designed to support:
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_netherite_sword.png" width="16"> Vanilla boss arenas
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_wither_skeleton_skull.png" width="16"> MythicMobs integrations
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_totem_of_undying.png" width="16"> Round-based combat
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_beacon.png" width="16"> Progression systems
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_nether_star.png" width="16"> Unlockable arenas
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_bell.png" width="16"> Cinematic gameplay

Unlike traditional arena plugins, BBattleASD was designed as a complete boss progression infrastructure capable of powering entire RPG and prison combat ecosystems.

<br>

The command architecture inside BBattleASD was designed to centralize arena management into a modern GUI-driven workflow.

Instead of relying on configuration files for every arena modification, almost every system can be configured directly in-game.

### <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_redstone_torch.png" width="18"> Arena Administration

Administrators can:
- create arenas
- edit rounds
- configure rewards
- modify mobs
- manage cooldowns
- configure unlocks
- synchronize spectator systems
- configure battle behavior

without requiring server restarts.

### <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_compass.png" width="18"> Independent Arena Ecosystem

Every arena behaves independently, allowing:
- different bosses
- different rounds
- different rewards
- unique progression
- custom cooldowns
- unique unlock requirements

for every battle.

</details>
